About The Webpage

For Lab 6, I decided to use version 3 of The Movie Database (TMDb) API.

The home page of my application allows you to use three of the API's features. You can search by movie, by TV shows, and by people using the API's Search Movies, Search TV Shows, and Search People features . After searching for a movie, TV show, or person, the application allows you to go through and view every result of your search query. Once you find the information you were looking for, the application shows you the more details about your search. For movies and TV shows, the application uses the API's features of Get Details, Get Credits, Get Similar.

For people, the application uses the API's features for Get Details and Get Combined Credits. All of this information is displayed to the user.

The discover page of my application allows you to discover trending, top rated, and coming soon movies and TV shows. It uses the API's Get Top Rated and Get Upcoming features, as well as the Get Trending feature. All of the information displayed to the user is provided by The Movie Database's API. The application uses AJAX and JQuery to implement its functionality.
