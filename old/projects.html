My Favorite Projects

Throughout my time at Temple University, I coded and designed a handful of projects. These are windows software applications, web applications, and mobile applications. Click on 'View Project on GitHub' to see the published code for each application!

Felix - Pet Adoption Website

This project was a pet adoption website where pet adopters could find an animal they're compatible with and shelters could post pets in their care. It gave me experience with building Web APIs (REST and SOAP) and building a safe multi-tier login and registration system. The registration/login system contained forgotten password recover, security questions and email validation.

Temple Bookstore

This project required me to build a database of books, create an order form for customers and a report for the bookstore's management. This project gave me experience building web applications that conform to the multiple-tier software architecture. It introduced me to Gridview controls and dynamic building using ASPX pages.

YAHTZEE - Windows Game

For this project, I designed and coded a simulation of the popular board game, Yahtzee. I designed many classes and functions from the ground up in order to record every turn in a two-player version of the game.

It gave me experience building a small scale Web Application that could be used in the real-world. I had to develop a data model with multiple databases and stored procedures for database access.

AllerGenius - Mobile Application

An application to easily manage your food allergies, intolerances and special diets on the go. AllerGenius gathers information from thousands of websites and tells you exactly what you can eat at the restaurant of your choosing. Designed for my Mobile Media course, and coded a small scale prototype using Angular 4.0 for Client-Side Scripting for the Web.
