What is Angular?

-Google created this OpenSource client side JS framework in 2010

-It is a single page application framework. It’s a form of javascript where written code can be easily tested by unit testing.

-The main concept for this application is so that the html page and the javascript logic can be developed at the same time which gives us faster productivity for unit testing.


Why was Angular 2 created?

- Renewal of the original Angular in 2.0 version in 2014..

- In this current framework, controllers and $cope have been removed and are replaced by components and directives.

- This is a framework for building complex application in the browser

Introduces a number of new features:
- Support for modern browsers and mobile development

- Moved many functionality into modules to make it easier to us ein third party libraries

- The use of TypeScript: TypeScript is a superset of JS and adds features like static typing and class based object-oriented programming
